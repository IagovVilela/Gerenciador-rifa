CONFIGURAÇÃO DO SUPABASE
========================

SUA PROJECT URL JÁ IDENTIFICADA:
https://qixtamnvrexfjmfuejfz.supabase.co

AGORA VOCÊ PRECISA:

1. Ir em Settings → API no Supabase
2. Copiar a "anon public" key (chave longa)
3. Colar aqui:

SUPABASE_URL=https://qixtamnvrexfjmfuejfz.supabase.co
SUPABASE_ANON_KEY=COLE_A_CHAVE_AQUI

4. Renomeie este arquivo para .env.local
